# CrowdSense Hyderabad - Smart Tourism Intelligence Website

A modern, responsive crowd monitoring and tourism planning web application for Hyderabad's iconic landmarks.

## Setup Instructions

1. Add Required Assets:
   - Place any landmark images in the project folder
   - Ensure all image filenames match those referenced in `script.js`
   - Images should be in the same directory as `index.html`

2. Open the Website:
   - Simply open `index.html` in your web browser
   - Or use a local server for better performance:
     ```
     python -m http.server 8000
     ```

## Features

- Responsive design that works on all devices
- Real-time crowd prediction for popular Hyderabad landmarks
- 7-day and 30-day crowd forecast charts
- Google Maps navigation from user's current location
- Detailed place profiles with timings, entry fees, and highlights
- OTP-based mobile login for personalized experience
- Admin portal to manage places and configure crowd alerts
- Modern gradient backgrounds and smooth hover effects

## Places Covered

- Golconda Fort
- Charminar
- Osman Sagar (Gandipet)
